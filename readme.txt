Enable & create Google+ Interactive Posts for your blog posts & pages.

== Description ==

Provides an easy to use form to created properly constructed and formatted Interactive posts on Google+.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `rh-metadata` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does this work? =

This plugin places a button about the Post/Page RTE (Next to the Add Media icon). Click on the button to fill in any of
the structured meta data necessary to make a Google+ post with a call to action (CTA). Default settings may be setup on
the plugin main configuration page. This is labeled in your left hand column as 'Markup'.

= I have a problem/question, what do I do? =

This plugin is maintained on [GitHub] (https://github.com/rankhammer/structured-data "Structured Data on GitHub"). You
can post your issues there, or contact us via [Twitter] (http://www.twitter.com/rankhammer "RankHammer on Twitter") or
on [Google+] (https://plus.google.com/u/1/112481538017885173789 "RankHammer on Google+").
